# MockCNData
Mock Chinese Data），专为生成中文测试数据

自用,由 AI 生成

```xml
<dependency>
    <groupId>cn.lacia.github</groupId>
    <artifactId>mockcndata</artifactId>
    <version>1.0.0</version>
</dependency>
```


### V1.0 基础功能：

- [x] 支持身份证、姓名、地址、营业执照等基本数据类型

- [x] 提供批量生成功能

- [ ] 完整单元测试覆盖

### V1.1 增强功能：

- [ ] 支持自定义地区分布

- [ ] 添加英文姓名和地址生成

- [ ] 增加数据导出功能（CSV、JSON）

### V1.2 扩展功能：

- [ ] 支持更多国家/地区的证件生成

- [ ] 添加企业数据生成（公司名、组织架构等）

- [ ] 支持自定义数据模板
